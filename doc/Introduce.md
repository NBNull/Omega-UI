# 介绍
> 基于React的响应式开发UI框架

### 特性
- 现代化的视觉UI设计，符合现代审美标准。
- 从基础元素到复杂交互，多元化的组件复用技术。
- 完全开源，丰富的使用示例。

### 浏览器要求
- 现代化浏览器（Chrome、Firefox较高版本，IE 10+）。
- 浏览器需支持Ecmascript 2015。
- 其他增强特性支持。
