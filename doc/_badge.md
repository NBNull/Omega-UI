## Badge组件

### 配置参数

Button组件的公共配置参数如下

| 参数名 | 类型 | 默认值 | 是否必须 | 说明 |
| ---- |---- | ---- | ---- | ----- |
| element | string | 'span' | 否 | 设置渲染元素类型
| color | string | 'info' | 否 | 设置显示状态颜色
| isWide | boolean | true | 否 | 设置是否以较宽的方式显示
| classList | array | null | 否 | 设置自定义的Class
| style | array | null | 否 | 设置自定义的Style
